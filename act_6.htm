Siglas

POP(Post Office Protocol)

: Protocolo que permite a los clientes de correo electrónico recuperar mensajes de un servidor de correo.

IP (Internet Protocol)

: Protocolo que se encarga de dirigir y enrutar los paquetes de datos a través de una red. Utiliza direcciones IP para identificar dispositivos.

OSI (Open Systems Interconnection)

Modelo de referencia que define cómo se comunican los sistemas de computadoras en redes. Tiene 7 capas: física, enlace de datos,red,transporte, sesión, presentación y aplicación.

TCP (Transmission Control Protoco)

:Protocolo de transporte orientado a la conexión que asegura que los datos lleguen completos y en orden a su destino.

UDP (User Datagram Protocol)

: Protocolo de transporte sin conexión, más rápido que TCP, pero no garantiza la entrega ni el orden de los paquetes.

SSH (Secure Shell)

: Protocolo seguro que permite la conexión remota a servidores, garantizando la privacidad y la autenticidad de los datos.

FTP (File Transfer Protocol)

: Protocolo utilizado para transferir archivos entre computadoras a través de una red TCP/IP.

SMTP (Simple Mail Transfer Protoco)

: Protocolo para el envío de correos electrónicos a través de Internet.

DHCP (Dynamic Host Configuration Protocol)

: Protocolo que asigna direcciones IP automáticamente a los dispositivos en una red.

DNS (Domain Name System)

: Sistema que traduce nombres de dominio (como www.ejemplo.com) a direcciones IP.

HTTP (Hypertext Transfer Protocol)

: Protocolo utilizado para la transferencia de páginas web y otros recursos a través de la red.

ARP (Address Resolution Protocol)

: Protocolo que mapea direcciones IP a direcciones MAC (físicas) en una red local.

MAC (Media Access Control)

: Dirección única asignada a cada dispositivo de red para identificarlo en la capa de enlace de datos

LAN (Local Area Network)

: Red de área local, que conecta dispositivos en una zona geográfica limitada, como una oficina o casa
